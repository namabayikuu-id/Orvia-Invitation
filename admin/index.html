📘 Panduan Setup Vercel Environment Variables
Set environment variables berikut di Vercel → Project → Settings → Environment Variables:
1
GITHUB_TOKEN — Personal Access Token GitHub dengan scope repo
Buat di: github.com/settings/tokens/new
2
GITHUB_OWNER — Username GitHub kamu, contoh: samueljkt
3
GITHUB_REPO — Nama repository, contoh: undangan-digital
4
ADMIN_SECRET — Password bebas untuk proteksi API admin, contoh: rahasia123
5
GITHUB_BRANCH — Branch utama repo, biasanya: main
